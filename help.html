Quick Start

  1. Open the extension popup and enter a target URL.
  2. Click Start. DevTrace resets the active session first, then begins capture.
  3. The current browser tab reloads to the target page so new requests can be observed.
  4. Watch requests arrive in real time, then filter, inspect, or export what you need for debugging.

Capture Modes

  • Main Domain Only: only the exact target host is tracked.
  • Include Subdomains: captures the target host and its subdomains.
  • All Domains (+ iframes): keeps the session broad for embedded resources.
  • Whitelist Mode: only listed hosts are accepted.

Export Behavior

  • Export JSON saves the selected downloadable resource URLs as a JSON array.
  • Export Resources opens a folder picker first, then writes files into a domain/path structure.
  • Already completed downloads stay marked as downloaded and are skipped on later export runs.
  • If direct folder writing is blocked, DevTrace falls back to the browser download pipeline.

Important Notes

DevTrace is designed as a developer network inspection tool. It captures requests that happen after capture begins and cannot retroactively reconstruct resources that finished loading before the listener was active.
  • Clicking Start automatically performs a stop and data reset for the active session first.
  • Clear All Data removes the current captured list and related persisted download markers for the active domain.
  • Reset Session stops capture, clears the active domain session, removes the saved URL, and returns the extension to a fresh stopped state.
  • Completed download status remains visible after popup reopen or page refresh.

Permissions and Privacy

  • The extension requests host access at runtime instead of hardcoding broad site access up front.
  • Captured request state and download markers are stored locally in the browser profile.
  • Exported resources are only written after explicit user action.
  • Product positioning should remain developer-facing: network inspection, API debugging, and resource analysis.
Limited Use disclosure: browser activity data is used only for DevTrace's user-facing developer inspection, API debugging, and resource analysis features.
